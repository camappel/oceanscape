Oceanscape: an Infinite Procedurally Generated Storm Sonification Machine

I wanted to represent the metocean conditions of the English Channel in an abstract, experiential way, and so I built an infinite procedural ambient music machine.

My machine does not use transformers, imagination, or any such human technology. Instead, I gave it a set of rules, which it is incapable of breaking, and real oceanographic data from the Channel Coastal Observatory (coastalmonitoring.org) API at Brighton and Pevensey Bay during 2024's Storm Bert. When the machine applies the data to the rules, it sounds a bit like music.

For example, the tides set the tonal centre, which is then outlined by pads. Ornamentation melodies are trigered by chance, and the probability is defined by the surge residuals. To get the undulating effect that's supposed to mimic the waves themselves, I assigned various filters and FX to atmospheric and wave data. Timesteps follow the tide record (10-minute resolution) and each channel is min–max normalized to 0–1.

The development of this machine was inspired by the music of Brian Eno, Bon Iver, Miles Davis, and the video game No Man's Sky.
